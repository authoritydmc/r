How to Create Shortcuts

Static Shortcut: Always redirects to the same URL.
  1. Click New Shortcut on the dashboard.
  2. Enter a unique shortcut pattern (e.g. google).
  3. Select Static as the type.
  4. Enter the target URL (e.g. https://www.google.com).
  5. Click Create Shortcut. Now, visiting /google will redirect to Google.
Dynamic Shortcut: Uses curly braces ({param}) in the target URL. The value is filled in from the URL path.
  1. Click New Shortcut on the dashboard.
  2. Enter a unique shortcut pattern (e.g. json).
  3. Select Dynamic as the type (or let the app auto-detect curly braces).
  4. Enter the target URL with a variable, e.g. https://jsonplaceholder.typicode.com/posts/{id}.
  5. Click Create Shortcut. Now, visiting /json/10 will redirect to https://jsonplaceholder.typicode.com/posts/10.
User Dynamic Shortcut: Uses square brackets ([param]) in the target URL. The value is prompted for and stored in your browser.
  1. Click New Shortcut on the dashboard.
  2. Enter a unique shortcut pattern (e.g. apikey).
  3. Select User Dynamic as the type (or let the app auto-detect square brackets).
  4. Enter the target URL with a variable, e.g. https://api.example.com/data?key=[apikey].
  5. For each [param], you will be asked for a description. This helps users know what to enter.
  6. Click Create Shortcut. The first time you visit /apikey, you will be prompted for your value, which is saved in your browser for future use.
