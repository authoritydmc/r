How to Create Dynamic Shortcuts

New: To create a shortcut, click New Shortcut on the dashboard. The shortcut pattern is now read-only by default—click the pencil icon to edit it. The app will check for duplicates and ensure the pattern is never empty.
Dynamic shortcuts use curly braces (e.g. {userId}) in the target URL. When someone visits /yourpattern/123, {userId} is replaced with 123.
  1. Click New Shortcut on the dashboard.
  2. Edit the shortcut pattern by clicking the pencil icon. Enter a unique name (e.g. jsonpost).
  3. Select Dynamic as the type (or let the app auto-detect based on curly braces in the target URL).
  4. Enter the target URL. For example:
    https://jsonplaceholder.typicode.com/posts/{postId}
  5. Save the shortcut. Now, visiting /jsonpost/1 will redirect to https://jsonplaceholder.typicode.com/posts/1.
Example: Dynamic shortcut for JSONPlaceholder POST API
Shortcut Pattern: jsonpost
Type: Dynamic
Target URL: https://jsonplaceholder.typicode.com/posts/{postId}
Usage: Visit /jsonpost/10 to go to https://jsonplaceholder.typicode.com/posts/10 (shows a fake post from the API).
