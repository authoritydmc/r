How to Create Dynamic Shortcuts

New: URL creation is now only available from the Edit page.
To create a shortcut, visit /edit/<shortcut> (e.g. /edit/meetwith).
Real-time suggestions: The edit page detects if your shortcut is Static or Dynamic and provides helpful hints. It will auto-prefix https:// if missing.
  1. Go to the edit page for your shortcut (e.g. /edit/meetwith).
  2. Type your target URL. Use curly braces (e.g. {name}) for a Dynamic shortcut; otherwise, it's Static.
  3. Let the form help you: It shows a green/blue box and auto-adds https:// if needed.
  4. Save the shortcut: Visiting /meetwith/raj will redirect to https://g.co/meet/raj if the target is https://g.co/meet/{name}.
Tip: You can use any variable name in curly braces (e.g. {user}, {id}). It substitutes the path segment after the shortcut.
